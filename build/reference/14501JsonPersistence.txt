@@tooltip Basic example showing how to use Umple's Json serialization/de-serialization feature

@@description
<p>JSON as a data-interchange format that can save and load data in many context and languages. For example, it can be used in transmitting over networks. and saving the state of the system persistently. </p>
<p>To better realize the functionality of Umple in terms of generating JSON, we would like any object to have:</p>
<ul>
<li><b>String toJson()</b> method that generates (serializes) a Json representation </li>
<li><b>Object fromJson(String)</b> static method that would instantiate (deserialize) the object and its related objects</li>
</ul>
<p>To generate functions for serializing and deserializing JSON, use command <i>umple [FILENAME].ump -g Java -s genJson -c-</i> to compile the code.</p>
&nbsp;<br/>
<p><b>These examples below</b>: demonstrate how use toJson() and fromJson() statements to serialize/de-serialize to/from Json.</p>
